CIS-255: Client Side Web Application Development

SUMMARY

  • Client Side Web Application Development teaches students how to conceptualize, design, code, test and deploy software that runs in a browser. Topics include: web concepts, HTML, CSS, BootStrap, JavaScript, jQuery, and AngularJS.

TEXTBOOK / ONLINE RESOURCES

  • Fundamentals of Web Development (2ed), covers most topics in the course, with supplemental materials available at: Companion Site (login) , Online Textbook (login) , and Amazon.com
  • Eloquent JavaScript, free online textbook, with sandboxes for editing and running code snippets,
  • The Missing Link: An Introduction to Web Development and Programming, a free online textbook , which is part of a Coursera course , called Web Applications for Everybody, with lots of free lesson materials
  • w3schools.com, a very popular site , and good reference for how to use code, with sandboxes for editing and running code snippets, and a YouTube channel presenting online demonstrations,
  • Derek Banas, a major league YouTuber with very cool, fast-moving programming tutorials,
